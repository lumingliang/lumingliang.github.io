seo 的相关技术

  1. 搜索引擎，用户输入关键词，然后搜索，然后百度第一页有多个排列，排在前面的会得到点击，进入后发现商品或者内容喜欢，则会购买商品，或者看广告，所以取得流量，网站就可以赚钱
  2. seo优化就是针对搜索引擎的算法，优化自己的网站的关键词，内容等，让搜索蜘蛛可以认定这是一个比较好的网站，自然会针对这个关键词，把它排名在前面
  3. 百度推广，利用给百度交钱，获得排行
  4. 灰帽，白帽，黑帽，就是利用seo引擎的模拟达到欺骗效果
